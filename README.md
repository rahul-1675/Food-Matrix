# Food Matrix 🍽️

Food Matrix is a responsive food ordering website built using HTML, CSS, Bootstrap, and JavaScript.

## Features

- 🍕 Multiple food categories
- 🔍 Search functionality
- 🛒 Shopping cart interface
- 📱 Responsive design
- 🔐 Sign-in page
- 🎨 Modern UI with Bootstrap

## Technologies Used

- HTML5
- CSS3
- Bootstrap 4
- JavaScript

## Project Structure


## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/rahul-1675/Food-Matrix.git
```

2. Open `mainpage.html` in your browser.
